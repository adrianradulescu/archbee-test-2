Here, the wizard will begin to run a connection check to validate all details entered in the previous steps and that all permissions are correctly set-up.

If there are no issues, a large green checkmark will appear to confirm that the connection has been established. Click on **Next Step** to begin matching bank accounts.

## Failed Connection Checks

If any permissions are missing, a red X will be displayed, as shown in the screenshot below, along with a clear, plain English explanation of the failure.
